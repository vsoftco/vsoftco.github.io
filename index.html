Professional information: I am a postdoctoral fellow at the Institute for Quantum Computing, working with Michele Mosca on theoretical aspects of quantum computation. I graduated from Carnegie Mellon University with a PhD in Theoretical Physics, under the supervision of Robert B. Griffiths. I am interested in fault-tolerant quantum processing, quantum architectures, quantum communication, cryptography (quantum and post-quantum) and query complexity. I am a referee for: Nature, Physical Review Letters, Physical Review A, Quantum Information Processing, Quantum Information and Computation, Physics Letters A, International Journal of Quantum Information, International Journal of Theoretical Physics, Entropy, Optics Communications.

Some code I wrote (complete GitHub repositories):

  • Quantum++ – a C++11 general purpose quantum computing library, composed solely of template header files. It uses the Eigen 3 linear algebra library and, if available, the OpenMP multi-processing library. Status: active, version 0.8.6, relased under GNU Public License.
  • pytexnumber – a Python 2/3 console program for renumbering latex references (labels).
    • Usage:
      • python3 py3texnumber.py - for the Python 3 version.
      • python py2texnumber.py - for the Python 2 version.
    • Description: replaces all labels in <in.tex> that match a given input <pattern>, such as \label{<pattern>...} and all corresponding references (\ref, \eqref, \pageref) with renumbered ones <replacement>idx, where idx counts the matching \label{<pattern>...} order of appearance, i.e. equals 1 for the first \label that matches the input pattern, 2 for the second etc. The file <in.tex> is left unchanged and the modifications are written to <out.tex>.
    • Example: consider a latex file <source.tex> having three labels, named, in order of appearance, \label{eqn5}, \label{eqn_important} and lastly \label{entropy}. Then the command
      • python3 py3texnumber.py in.tex out.tex eqn Eqn
      replaces \label{eqn5} by \label{Eqn1} (and also all references to eqn5 automatically become references to Eqn1) and replace \label{eqn_important} by \label{Eqn2} (again together with the corresponding references). The \label{entropy} remains unchanged since it does not match the input pattern \label{eqn...}. The results are saved to the file <out.tex>.
  • cpptexnumber – the C++11 version of pytexnumber above. A C++98 version is also available.
    • Usage: same as pytexnumber above.
  • GTexnumber – a GUI Java program for renumbering latex references (labels). Texnumber: the console version. The Java sources are included in the .jar files. The Netbeans projects are located here.
    • Usage: Unzip the archive then run the program with
      • java -jar GTexnumber.jar
      for the GUI version, or
      • java -jar Texnumber.jar
      for the console version.
    • Description: same as pytexnumber above. For the GUI version, the user's input is performed through the graphical interface.
